# Claude Desktop Skills

A collection of 10 high-leverage skills for Claude Desktop — reusable, structured prompts that turn Claude into a focused tool for common knowledge-work tasks.

Each skill is a single markdown file you can drop into Claude Desktop's skills directory (or paste into a conversation) to get consistent, repeatable results.

## The skills

| Skill | What it does |
|-------|--------------|
| `research-synthesis` | Pull sources into a clear, cited synthesis |
| `decision-matrix` | Weigh options against criteria and recommend |
| `meeting-to-actions` | Turn messy meeting notes into owned action items |
| `message-crafter` | Draft clear, well-toned messages and emails |
| `feedback-synthesis` | Consolidate scattered feedback into themes |
| `document-diff` | Compare two documents and summarize what changed |
| `learning-path` | Build a structured path to learn a new topic |
| `content-transform` | Reshape content between formats and audiences |
| `explain-at-level` | Explain a concept at a chosen depth or audience |
| `structured-brainstorm` | Generate and organize ideas systematically |

See `ALL-SKILLS.md` for the full set in one place.

## Usage

1. Open any skill file (e.g. `decision-matrix.md`).
2. Add it to Claude Desktop as a skill, or paste its contents into a chat.
3. Provide your inputs and let Claude run the skill.

## Why

These are the tasks I found myself repeating with Claude — so I turned each into a reusable, battle-tested prompt instead of rewriting it every time. Fork it, adapt the skills to your own workflow, and add your own.
